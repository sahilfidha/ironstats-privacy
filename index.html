IronStats Privacy Policy

Last Updated: May 2026

Overview

IronStats is a workout tracking application designed to help users log workouts, track progress, manage routines, and optionally interact with friends through social features.

Information Stored Locally

The following information is stored locally on the user's device:

  • Workout history
  • Exercises
  • Sets, reps, and weights
  • Workout notes
  • Exercise notes
  • Form check information
  • Routines and templates
  • Personal training statistics
  • Application preferences

Information Stored in Firebase

If social features are used, IronStats stores the following information using Firebase:

  • Email address used for authentication
  • User identifier (UID)
  • Display name
  • Friend relationships
  • Friend requests
  • Routine sharing requests
  • Shared routine templates
  • Shared statistics summaries when stat sharing is enabled

Information Not Uploaded

IronStats does not upload the following information as part of its current social features:

  • Complete workout history and detailed training logs
  • Individual workout logs
  • Set-by-set workout records
  • Workout notes
  • Exercise notes
  • Raw archive contents

Sharing Features

Users may remove friends, delete shared content, clear workout history, reset local application data, or permanently delete their account using controls provided within the application.

Data Deletion

Users may remove friends, delete shared content, clear workout history, or reset local application data using controls provided within the application.

Third-Party Services

IronStats uses Firebase services provided by Google for authentication, database storage, and social functionality.

IronStats may update this Privacy Policy periodically as features evolve or services change.
